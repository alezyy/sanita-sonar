 1 - Présentation
 ==========================================================

 Ce logiciel est destiné à la gestion de réclamation. C’est 
 un outil qui automatise le processus de traitement d'une 
 réclamation d'assurance santé.

 Il sera lancé via IntelliJ en recevant en paramètre quatre 
 fichiers respectivement correspondant au fichier d’entrée, 
 au fichier des polices, au fichier historique et au fichier 
 resultat.

 Dans le cadre de ce travail, l’interface usagé ne sera pas 
 implementé.
 

 2 - Technologies utilisées
 ==========================================================

 - IDE IntelliJ
 - Fichiers E/S en JSON
 - Maven 3.6.3
 - Java 11
 - Java(TM) SE Runtime Environment (JDK15);
 - JUnit 4.0
